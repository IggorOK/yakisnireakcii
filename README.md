🎮 Як це працює
Ви бачите штатив з пробірками, в яких — невідомі розчини. На столі — 20 реактивів (індикатори, кислоти, луги, солі, спеціальні реагенти). Ваша задача:

Обрати реактив зі столу
Натиснути на пробірку — побачити ознаки реакції (осад, бульбашки, зміна кольору, запах)
Записати спостереження (журнал ведеться автоматично)
Підписати пробірки — визначити, яка речовина де знаходиться

Оцінка залежить від кількості використаних реактивів — чим менше, тим краще!

🚀 Запуск
Один HTML-файл, ніяких залежностей для збірки:
bash# Варіант 1: просто відкрити файл
open chem-detective.html

# Варіант 2: через HTTP-сервер
python3 -m http.server 8000
# → http://localhost:8000/chem-detective.html
Для GitHub Pages: перейменуйте файл на index.html, увімкніть Pages у налаштуваннях репозиторію.
